Getting Started
------------------------
`GooglePlacesAutoComplete` is a small kotlin demo for auto search in google map.

TypeFilter
------------------------
ADDRESS,
CITIES,
ESTABLISHMENT,
GEOCODE,
REGIONS

Change this filter :
- open MapExtention
- getAutocomplete Method
- setTypeFilter(TypeFilter.CITIES)
